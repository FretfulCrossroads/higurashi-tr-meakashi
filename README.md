# Chapter 5 - Meakashi
## Turkish Translation

🇬🇧 English | 🇹🇷 [Türkçe](README_TR.md)

This repository contains the Turkish fan translation for **Meakashi**, the fifth chapter of the *Higurashi When They Cry Hou* series.

> [!WARNING]
> This is an AI-assisted fan translation.
>
> Dialogues and monologues were generated using Large Language Models (LLMs), then processed and reviewed before release.
> Translation errors and unnatural wording may still exist.

---

# Installation

1. Own a copy of the game 
2. Download the latest zip release of translation from **Releases** tab.
3. Extract the files and replace the files in the "Higurashi When They Cry Hou - Ch. 5 Meakashi\HigurashiEp05_Data" folder with the exact same ones.
